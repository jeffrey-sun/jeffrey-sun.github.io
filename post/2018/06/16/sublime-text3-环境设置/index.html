引言

今天电脑入域，强制切换用户了，之后这个笔记本很久没有开，什么环境也没有。想着最近可能还是需要在家看点东西，因此折腾一下环境。听说 VScode不错，弄了半天，还是不怎么顺心。因此还是打算折腾一下 稍微熟悉点的 sublime text（机器比较老，担心IDE跑不动，特别jetbrains 家的）。

环境说明

类型 数据
系统 Windows7
软件版本 V3.1.1

插件安装

安装Package Control

这个好像是在tools 还是哪个地方有个菜单，点击一下就执行了。这个和原来有区别，原来需要自己执行命令。

安装汉化插件 ChineseLocalization

按住 shit+ctrl+p,出现命令输入框，输入pcip(即：Package Control： Install Package的简写)，出现上面的命令。 回车执行（那个心态平活点，可能由于网络原因新的命令出来的比较慢）。输入 hineseLocalization 找到对应的插件就好。 其实不汉化也没有什么的，像我这样的英语渣子，用起来问题不大。

PS： 后面就不再介绍插件安装命令了，都一样了。 因为我感觉， 长时间不用我会忘记命令的， 但是了解一次后面应该是没有什么问题的。 后面的就是介绍一下插件的作用和配置。

DocBlockr

DocBlockr 是一个代码快注释插件。支持的语言有： JavaScript (including ES6), PHP, ActionScript, Haxe, CoffeeScript, TypeScript, Java, Groovy, Objective C, C, C++ and Rust.

BracketHighlighter

BracketHighlighter 是一款Sublime下匹配标签高亮的小插件，可以把匹配到的如 {}、()、”、””等对应的符号或者标签高亮显示。

ConvertToUTF8

ConvertToUTF8 用来解决文件编码问题的。

git

git 可以在命令里执行git操作。注意这个需要配置一下，git_command等。

结尾

目前就装了这几个插件，其他的看具体的需求。后面如果有会继续更新。
