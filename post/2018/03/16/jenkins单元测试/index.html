jenkins插件安装

为了能够从xml中获取Unit test 的结果，可以安装 Publish Junit test result report 组件。使用时需要制定xml文件位置， 可以使用*进行匹配

不同语言生成单元测试xml

java

maven 构建单元测试xml

maven 构建的时候如果有JUnit用例时，xml记过默认会在target目录的 surefire-reports 子目录下面。 在编码过程中，maven需要配置 JUnit 依赖，写测试用例即可

Python

unittest 单元测试生成 xml

pyhthon 标准库自带 unittest 模块，为了能够把单元测试结果保存到 xml中，环境需要安装 unittest-xml-reporting组件。 该组件支持 unittest 和 Doctest 两种单元测试方式。
