背景

最近之前的网络主机到期了，换了一个服务商。想写点python的工具，内置的python版本是2.7。 因此在这里记录过程

环境说明

操作系统： centos7

安装 Python

使用yum工具进行安装

1
2
3
4
5
6
7

# 查看yum源上有那些可用的python3版本
yum search python3 

# 在我的机器上执行命令后有3.4和3.6两个版本可以安装。这里我选择 3.6版本
yum install python36
# 安装pip3 包管理工具
yum install python36-pip

安装虚拟环境管理工具

1
2

# 执行pip3安装 pipenv
pip3 install pipenv

若是没有pip3命令，可以先按照上一节的说明安装
