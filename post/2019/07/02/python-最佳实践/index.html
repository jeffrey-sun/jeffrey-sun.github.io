命令行库简单分析：

  • Clint 最新支持 python 3.4， 目前库开发人员已经将仓库存档，后续不再维护，可以跳过
  • Click 通过装饰器完成，使用时代码配置非常少，简单应用推荐
  • docopt 是一个轻量级，高度Pythonic风格的包，它支持 简单而直觉地创建命令行接口，它是通过解析POSIX-style的用法指示文本实现的。使用稍显复杂，功能强大。
